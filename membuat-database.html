Belajar Database MySQL

Membuat Database dapat dilakukan dengan 2 cara :

  1. GUI

  2. CLI

GUI

  1. Klik "Baru" pada halaman utama

  2. Berikan nama database lalu klik "Buat"

    3. Hasil Pembuatan Database

  3. Buat Tabel. Masukkan nama tabel dan jumlah kolomnya

  4. Buat kolom dengan ketentuan sebagai berikut:

    • "id" adalah kolom dengan tipe data Integer berukuran 5 yang berperan sebagai Primary Key, aktifkan Auto Increment supaya kolom id dapat diisi berurutan secara otomatis
    • "nama" adalah kolom dengan tipe data Varchar berukuran 30
    • "tanggal_lahir" adalah kolom dengan tipe data Date
    • "jenis_kelamin" adalah kolom dengan tipe data Varchar berukuran 10
    • "alamat" adalah kolom dengan tipe data Varchar brukuran 50

    Hasil Pembuatan Tabel

CLI

  1. Buka Command Prompt

  2. Arahkan ke direktori mysql/bin pada folder XAMPP

  3. Ketik perintah "mysql -u root"

    4. Kita akan login dengan menggunakan user root

  4. Ketik perintah CREATE DATABASE belajar_sql;

  5. Buat Tabel "biodata". Ketik perintah :

    6. CREATE TABLE biodata (
    id INT(5) PRIMARY KEY AUTO_INCREMENT,
    nama VARCHAR(30),
    tanggal_lahir DATE,
    jenis_kelamin VARCHAR(10),
    alamat VARCHAR(50)
    );

    Tabel berhasil dibuat

  6. Cek struktur tabel "biodata". Ketik perintah DESC biodata;
