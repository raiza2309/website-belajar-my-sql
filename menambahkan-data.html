Belajar Database MySQL

  1. Pilih Database belajar_sql dan tabel biodata

  2. Klik menu SQL

  3. Ketikkan perintah berikut :

    4. INSERT INTO biodata (nama, tanggal_lahir, jenis_kelamin, alamat)
    VALUES ("nama_anda", "1999-12-31", "jenis_kelamin_anda", "alamat_anda");

    Data berhasil ditambahkan

  4. Tampilkan data. Ketik perintah :

    5. SELECT id, nama, tanggal_lahir, jenis_kelamin, alamat
    FROM biodata;

    Atau

    SELECT * FROM biodata;
